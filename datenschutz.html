Der Schutz deiner persönlichen Daten ist mir wichtig. Diese Datenschutzerklärung informiert dich darüber, welche Daten bei der Nutzung dieser Website und der GenAudioBook-App erhoben werden — und welche nicht.

Kurzfassung: Diese Website verwendet keine Cookies, kein Tracking und keine Analyse-Tools. Die App läuft 100% offline und sendet keine Daten an Server.

3. Website (genaudiobook.com)

3.1 Hosting

Diese Website wird über GitHub Pages gehostet, einen Dienst der GitHub, Inc. (88 Colin P Kelly Jr St, San Francisco, CA 94107, USA). Beim Aufruf der Website werden automatisch technische Daten (z.B. IP-Adresse, Browsertyp, Zeitpunkt des Zugriffs) in Server-Logfiles gespeichert. Dies geschieht durch GitHub und liegt außerhalb meiner Kontrolle.

Weitere Informationen: GitHub Privacy Statement

3.2 Cookies

Diese Website verwendet keine Cookies. Es wird lediglich die Spracheinstellung (DE/EN) im Local Storage deines Browsers gespeichert. Local Storage ist kein Cookie und wird nicht an Server übertragen.

3.3 Analyse & Tracking

Diese Website verwendet keine Analyse-Tools (kein Google Analytics, kein Matomo, keine Tracking-Pixel). Es werden keine Nutzungsprofile erstellt.

3.4 Google Fonts

Diese Website nutzt Google Fonts zur einheitlichen Darstellung von Schriftarten. Beim Aufruf der Seite wird eine Verbindung zu Servern von Google LLC (1600 Amphitheatre Parkway, Mountain View, CA 94043, USA) hergestellt. Dabei kann deine IP-Adresse an Google übermittelt werden.

Weitere Informationen: Google Datenschutzerklärung

3.5 Externe Links

Diese Website enthält Links zu externen Diensten (Gumroad für den Kauf der Pro-Version). Beim Klick auf diese Links gelten die Datenschutzbestimmungen des jeweiligen Anbieters.

4. GenAudioBook App

4.1 Lokale Verarbeitung

Die GenAudioBook-App läuft vollständig lokal auf deinem Mac. Alle Bücher, Audiodateien und KI-Modelle werden ausschließlich auf deinem Gerät verarbeitet und gespeichert. Es werden keine Daten an externe Server gesendet.

4.2 Lizenzaktivierung

Bei der Aktivierung der Pro-Version wird dein Lizenzschlüssel einmalig an die Gumroad API (gumroad.com) gesendet, um die Gültigkeit zu prüfen. Dabei werden folgende Daten übermittelt:

  • Der eingegebene Lizenzschlüssel
  • Die Produkt-ID (Permalink)

Es werden keine weiteren persönlichen Daten übermittelt. Die Lizenzprüfung erfolgt über eine verschlüsselte HTTPS-Verbindung.

Weitere Informationen: Gumroad Privacy Policy

4.3 KI-Modell Downloads

Beim erstmaligen Installieren einer KI-Engine werden die Modell-Dateien von HuggingFace (huggingface.co) oder PyPI heruntergeladen. Dabei kann deine IP-Adresse an diese Dienste übermittelt werden. Nach dem Download ist keine Internetverbindung mehr erforderlich.

4.4 Gespeicherte Daten

Die App speichert folgende Daten lokal auf deinem Mac:

  • Lizenzstatus und Nutzungszeit (verschlüsselt in ~/.genaudiobook_data)
  • Generierte Audiodateien (in deinem gewählten Export-Ordner)
  • Heruntergeladene KI-Modelle (in ~/Documents/GenAudioBook/Models/)
  • Log-Dateien (in ~/Documents/GenAudioBook/Logs/)

Alle diese Daten können jederzeit von dir gelöscht werden.
